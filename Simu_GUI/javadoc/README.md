# Cafeteria Simulator API Documentation

This directory contains the complete Javadoc API documentation for the Cafeteria Simulator project.

## Viewing the Documentation

To view the documentation, open `index.html` in your web browser. This will provide you with:

- **Overview**: Complete overview of all packages and classes
- **Package Summary**: Detailed information about each package
- **Class Documentation**: Full documentation for each class, interface, and enum
- **Method Documentation**: Detailed descriptions of all methods with parameters and return values

## Documentation Structure

The documentation is organized by package:

- **simu.framework**: Core simulation framework classes (Engine, Clock, Event, etc.)
- **simu.model**: Simulation model classes (Customer, ServicePoint, MyEngine, etc.)

## Generated Documentation

The documentation was generated using the standard JavaDoc tool and includes:

- Class and interface descriptions
- Method documentation with `@param` and `@return` tags
- Field documentation
- Package summaries
- Class hierarchy trees
- Search functionality
